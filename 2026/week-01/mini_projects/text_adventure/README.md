# Text Adventure (CLI)

A beginner-friendly text-based adventure game in Python.

## How to run

From the `text_adventure/` folder:

```bash
python src/main.py

Commands

    look -> describe the current room

    go <direction> -> move (north/south/east/west/up/down or n/s/e/w/u/d)

    take <item> -> pick up an item in the room

    inventory -> show what you're carrying

    help -> show commands

    quit -> exit the game

Goal

Find the key and reach the gate to win.
```
