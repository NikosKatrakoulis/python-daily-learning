# Mini Calculator

A simple calculator mini project written in Python.

## Features

- Basic arithmetic operations
- Stores calculation history in JSON
- Clean project structure

## Project Structure

- `main.py` — CLI menu (entry point)
- `calculator.py` — operations, calculator dispatcher, history manager
- `history.json` — created automatically after the first calculation

## How to run

From the project folder:

```bash
python main.py
```
